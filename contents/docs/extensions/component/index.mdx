## 虚拟列表

### gh-kL/cocoscreator-list

#### 介绍

这是一个基于 Cocos Creator 写的虚拟列表组件。本组件是配合 Cocos Creator 本身的滚动窗结构去写的，所以在编辑器中操作会很方便，所见即所得。

本组件支持：

-   所有类型布局。（单列、单行、网格，甚至各种花式 RIGHT_TO_LEFT、BOTTOM_TO_TOP）
-   分帧渲染。
-   循环列表。
-   选择模式。（单选、多选）
-   滑动模式。（普通、粘附、分页）
-   动态 Item 宽/高。（用来做聊天列表再适合不过）
-   动态删除 Item 项。
