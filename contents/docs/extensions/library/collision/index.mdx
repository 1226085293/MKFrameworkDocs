## cc-sat

### 介绍

碰撞检测可分为 Broad Phase （粗略检测）与 Narrow Phase （精细检测） 两个阶段。
SAT(Separating Axis Theorem)分离轴定理，适用于任意 2D 凸多边形的 精细检测 阶段。

## cc-sap

### 介绍

碰撞检测可分为 Broad Phase （粗略检测）与 Narrow Phase （精细检测） 两个阶段。
