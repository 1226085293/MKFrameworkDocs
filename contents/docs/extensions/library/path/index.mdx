## cc-astar

### 介绍

可以保证各平台计算一致性(可用于帧同步)

A 星巡路模块，支持：

四边形的 4 方向、8 方向及自由模式(基于 8 方向+漏斗算法)巡路
六边形的 6 方向巡路
同时还导出了一个用于路径平滑的函数(不能保证计算一致性)。
