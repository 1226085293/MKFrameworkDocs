## behavior3-ts

### 介绍

用途
Behavior3TS 是 Behavior3 库的原始实现和官方 TypeScript 版本。它提供结构和算法，帮助您为游戏或应用程序创建智能代理。

主要特点
基于(Marzinotto et al., 2014)的工作，他们提出了行为树的正式、一致和通用的定义；

-   优化控制多个代理：您可以使用单个行为树实例来处理数百个代理；

-   它被设计为以 JSON 格式加载和保存树，以便在多种环境、工具和语言中使用、编辑和测试它；

-   一个很酷的可视化编辑器，您可以在线访问；

-   库中提供了多个复合节点、装饰节点和动作节点。您仍然可以定义自己的节点，包括复合节点和装饰节点；

-   完全免费，核心模块和可视化编辑器均在 MIT 许可下发布，这意味着您可以将它们用于您的开源和商业项目；

-   轻量级的！
