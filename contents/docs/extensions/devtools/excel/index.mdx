## luban

### 介绍

鲁班是一个强大、稳定、稳定的游戏配置解决方案。其设计目标是满足从小型到超大型游戏项目的简单到复杂的游戏配置工作流需求。

luban 可以处理丰富的文件类型，支持主流的语言，可以生成多种导出格式，支持丰富的数据检验功能，具有良好的跨平台能力，并且生成极快。luban 有优雅的生成表格设计，支持良好的规则和插件化，开发者进行二次开发。开发者很容易就能将 luban 配置到自己的配置格式，定制出满足项目要求的强大的配置工具。

鲁班标准化了游戏配置开发工作流程，可以极大提升策划和节目的工作效率。
