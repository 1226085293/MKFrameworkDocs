欢迎使用 **MKFramework**! MK 框架是用于 CocosCreator3.x 的通用游戏框架。MK 框架在每个模块都有其独特的创新，并跟随引擎的最新版本更新。

## 开源

MKFramework 致力于帮助到更多人快速开发游戏，所以框架 **永久开源 🎉**，MK 框架不会对框架模块、分版本进行收费，也不会对其教程收费。后续可能会出一些付费的框架增强插件用于支持框架维护。也欢迎进行捐赠支持开源项目维护工作。

## 项目概览

| **模块** | **描述**                                                                                      |
| -------- | --------------------------------------------------------------------------------------------- |
| 音频     | 提供了微信/通用音频系统，支持音频多分组管理                                                   |
| 新手引导 | 多引导管理，多边形遮罩，支持任意步骤跳转 ，可任意插入/删除/添加步骤，一步骤一脚本             |
| 多语言   | 提供多语言文本\图片\节点。可动态增加文本/图片数据，支持编辑器预览！                           |
| 模块系统 | 提供单脚本、MVC 架构。成熟的 UI 管理器，完善的生命周期和展示层级控制                          |
| 网络     | 支持 Http、Websocket。可自定义编解码，收/发消息简化，可扩展收发数据结构                       |
| 资源     | 提供本地/远程/编辑器的资源加载与释放、创新的 Bundle 热更、Bundle 管理器机制                   |
| 数据     | 数据共享器（异步/同步支持），本地数据存储器（类型安全），数据监听器（可扩展 MVVM 或本地调试） |
| 任务     | 可重复利用的 Promise，以及顺序执行的任务管线                                                  |
| 事件     | 参数类型安全！可创建多实例事件对象，增加事件请求方法等待事件回调返回                          |
| 日志     | 支持多实例，打印等级控制，增加错误日志上传功能                                                |
| 适配     | 提供屏幕自动适配和节点适配                                                                    |
| 其他     | 提供节点粒度的暂停/恢复                                                                       |

## 项目特点

- **类型安全**：所有框架源码皆是基于类型安全设计

- **便于修改**：可像用户脚本一样直接修改框架 TS 源码

- **模块剔除**：框架模块支持手动剔除和恢复（TreeShaking）
