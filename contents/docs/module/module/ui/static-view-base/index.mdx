`mk.StaticViewBase` 是静态视图基类，无法被 `mk.uiManage` 管理，继承自 [`mk.LifeCycle`](./life-cycle)

## 何时被使用

如果符合以下条件，你可以使用 `mk.StaticViewBase`

-   不需要使用 `mk.uiManage` 管理当前模块

-   不想要 ViewBase 在属性面板上的展示

-   想要生命周期函数被正确调用

## 注意事项

`mk.StaticViewBase` 屏蔽了 `mk.Layer` 功能，如果你需要它，可以在你的类型中定义 `_isUseLayer = true;`
如下

```ts
class XXX extends mk.StaticViewBase {
    protected _isUseLayer = true;
}
```
