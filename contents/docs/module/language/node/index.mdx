## 介绍

多语言节点是一个组件，可以挂载任意节点上，用于展示不同的语种下展示不同的节点，或者适配 Layout 组件节点

![多语言节点](/docs/a30487e3-354b-48a7-abff-3f8b337eed7c.png)

## 使用

![节点](/docs/20465693-b799-4c6f-9369-50a11a5d02ee.png)

例如图中的 cn 节点为中文，en 节点为英文。则可以把 `MKLanguageNode` 组件挂载到它们的父节点上，
然后分别设置属性面板上的 "当前语言节点"

## 属性面板

### 语言

和 "当前语言节点" 配置使用，代表了 "当前语言节点" 在什么语种下展示

### 当前语言节点

在属性面板的 "语言" 下展示的节点

### layout 适配

你可以挂载到 Layout 组件节点上，组件会根据语种配置调整 Layout 的 `horizontalDirection`（水平排列子节点的方向）属性

## 成员

### currentNode

当前语言节点，未设置则为 null
